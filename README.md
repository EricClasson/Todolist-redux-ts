# React + TypeScript + Vite

Todolist created with react and typscript. I created a redux store to handle the function of the todolist.

### Built With

- Html
- React
- Typescript
- Tailwind
- Redux

## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run project
   ```sh
   npm run dev
   ```
